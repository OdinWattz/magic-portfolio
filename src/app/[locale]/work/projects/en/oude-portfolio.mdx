## Het portfolio

Voor mijn allereerste opdracht die ik gingen maken op mijn opleiding was een portfolio, dit portfolio was dus sinds vorig jaar niet meer veranderd alleen dingen toegevoegd.

## Hoe ging ik te werk?

- Als eerste had ik wireframes gemaakt voor mijn portfolio met behulp van Figma, in die tijd is het design wel echt 10x veranderd, omdat ik totaal niet blij was met mijn design totdat ik kwam op dit design.
- Na het maken van de designs en wireframes was ik begonnen met het programmeren van de website, op dat moment had ik amper kennis en ervaring met HTML en CSS.
- Toen ik dus eindelijk was begonnen moest ik nog veel opzoeken, omdat ik er nog geen kennis van had was het moeilijk.
- Tijdens dat ik bezig was had ik een plan met stappen voor welke pagina's ik graag als eerst wou maken en wat voor dingen ik wou toevoegen.
- Toen ik dat plan had gemaakt kon ik makkelijk zien wat ik wou doen en kon ik gemakkelijk aan het werk en dat merkte ik ook, omdat ik niet elke keer nog hoefde te bedenken wat ik nou wou doen.

## Dingen die ik heb gebruikt

- **HTML**: Hiermee kon ik de componenten toevoegen aan mijn portfolio.
- **CSS**: Met behulp van de CSS kon ik de website designen, Ik deed het toen nog erg inefficiënt en had ik voor elke pagina een eigen CSS-file, maar nadat ik werd beoordeeld heb ik dat nog veranderd naar 1 File.
- **Figma**: Hiermee kon ik de designs en wireframes maken.

## Waar liep ik tegenaan en wat heb ik er van geleerd?

Nou ik was begonnen met amper kennis dus in het begin vond ik het erg lastig, maar toen ik eenmaal bezig was viel het wel mee en kon ik mijn portfolio maken. Ik liep wel aan tegen die ID's en benamingen van mijn CSS classes en daarvoor was ik dus afgerekend en moest ik dat nog veranderen, achteraf heb ik dus de benamingen veranderd en alles in 1 geörganiseerde CSS file geplaatst.

## Wat is er nou gelukt

Dankzij deze opdracht heb ik een portfolio kunnen maken en kon ik daar alle dingen opzetten die ik op school zou maken. Voor dit portfolio heb ik een Voldoende gekregen. Zelf was ik toen ook tevreden met mijn website maar nu een jaar later vind ik hem erg gemakkelijk en heb ik besloten een nieuwe te maken. Hiervoor heb ik wel een al bestaand design voor gebruikt, omdat ik zelf designen nog lastig vind maar heb nu wel veel veranderd en een stuk mooiere portfolio.
